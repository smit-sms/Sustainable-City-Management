Build package.
> python setup.py sdist bdist_wheel

Install package from wheel file.
> pip install dist/etl_pipeline_ggn1_ase_g5-0.0.2-py3-none-any.whl --force-reinstall

Run tests.
> python test_main.py

Upload package.
> twine upload dist/*

Run pipeline.
> python -m etl_pipeline_ggn1_ase_g5 --db-name dbetl --db-path ./tests --logs-dir ./tests/logs
> etl_pipeline --db-name dbetl --db-path ./___TestsETLPipeline --logs-dir ./___TestsETLPipeline
